Bowser's Last Stand

Made with Bowser Engine, this was the last game I made with Game Maker. This was at an awkward time when I was transitioning to advocating free/libre software. As such, I did release my code as free/libre software (I opted to dedicate it to the public domain), but I didn't go the full mile and refuse to include proprietary code. Unfortunately, ENIGMA is unable to compile the source file since it was corrupted by Game Maker at some point.

On its own merits, this is a decent, but not outstanding platformer. I liked it enough that I'm planning on developing an ethical sequel to it.
