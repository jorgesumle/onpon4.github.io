INVADERS

INVADERS is a Space Invaders clone written in the BASIC-like language that comes on TI-8x calculators. It is under the GPLv3 (and I also grant permission to use any later version of the GPL published by the Free Software Foundation), but the choice to use the TI-8x interpreted language means it depends on these calculators' proprietary operating system and interpreter.

To be clear, although the source code is difficult to read, it is the real source code; I learned to avoid anything not syntactically necessary (like closing parentheses).
