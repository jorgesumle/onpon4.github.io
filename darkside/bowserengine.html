Bowser Engine

In 2009, I quit game development for a few months, for a stupid reason that I attribute to teenage hormones. However, this brief hiatus in game development for me ended when I saw Hello's Mario engine. It inspired me to make a "Bowser engine" which could theoretically be used to make "day in the limelight" games for Bowser. The result of this effort was "Bowser Engine".

My code in Bowser Engine was released publicly, and my code is in the public domain. Unfortunately, it depends on the proprietary Game Maker engine to work. It doesn't work with ENIGMA because the file got corrupted at some point.

I used this to develop Bowser's Last Stand.
