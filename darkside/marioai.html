Mario AI

Mario AI was a simple game I made to practice writing artificial intelligence (A.I.). Quite simply, Mario tries to stomp as many of an infinite army of goombas as possible. An A.I. could take control of Mario and attempt this task.

While not exceptional, the A.I. was able to perform really well. The highest score I was ever able to achieve was 1440, but I recorded a case of the A.I. getting 2315 points.

I think I did release the source code to this, but it was still made with the proprietary Game Maker game engine. It's possible that it would compile in ENIGMA today; that likely depends on whether or not the GMK file was corrupt (which they often are).

All downloads of Mario AI were hosted on file hosts that have since been shut down, so the game no longer exists.
