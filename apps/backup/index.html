Backup script

Category:Utility
License:Apache License v2
Description:Script which archives and encrypts important files for backup purposes.

This is a Python 3 script I rigged up to make regular backups easier to do. It uses a list of file names to determine what to back up, and puts them all in a tar archive before encrypting them with GnuPG.
