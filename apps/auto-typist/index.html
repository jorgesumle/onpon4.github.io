Auto Typist

Status:Complete
Category:Utility
License:GNU GPLv3+
Dependencies:Python 2.6 or later, but earlier than Python 3
Pygame 1.8 or later
Platforms:Platform-independent
Description:Automatic typist which can type at any speed you want.

This is a simple program I wrote originally so I could make simple videos without a microphone. It accepts some text and types it. While the typing happens, typing sounds are played to make it feel more realistic. It is very configurable, including allowing you to change how the text looks and how fast the typing is.
