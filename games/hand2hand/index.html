Hand2Hand

Status:Scrapped
Genre:Action
Perspective:First-person
Players:2
License:GNU GPLv3+

Hand2Hand was an idea I had for a 2-player action game. The idea was to have a game where the players punch and block each other's punches with pairs of analog sticks, each representing a hand. Unfortunately, it was an unrealistic project. I never got past buiding a basic, barely-working engine.

One of the biggest problems was the sheer amount of graphical resources that would be needed for the idea to work. I think the idea was flawed in general.

One interesting thing about this project was that I delved into the idea of creating a higher-level logic system in the code. I used video game consoles as an analogy. This is not quite the same idea as the SGE Game Engine, but it's a precursor to it.
